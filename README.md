# Pubsub

## Installation

```bash
npm i --production
npm i ava -g
```

## Quickstart

### Broker

Start/stop the broker:

```bash
npm start
npm stop
```
Start in debug:

```bash
npm run dev
```

### Subscribe

Subcribe to topics:

```bash
npm run sub topic1 topic2 topic3
```

### Publish

Publish to topic:

```bash
npm run pub topic1 "payload"
```
